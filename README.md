# CCF 大模型能力认证课程（中小学版）

> 基于 CCF 大模型能力认证大纲 · 中小学组知识点表

## 📖 课程简介

本课程专为中小学生设计，涵盖人工智能与大模型的核心基础知识，帮助学生系统性地了解 AI 世界，培养计算思维和科技创新意识。

## 🎯 学习目标

- 理解人工智能的基本概念和应用
- 掌握大语言模型的基础原理
- 学会使用 AI 工具解决实际问题
- 培养科技伦理意识和安全观念
