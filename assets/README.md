## 使用说明

### 在 Markdown 中嵌入图片

```markdown
![图片说明](./assets/images/filename.png)
```

### 嵌入外部图片

```markdown
![图片说明](https://example.com/image.png)
```

### 嵌入视频链接

```markdown
📺 [视频标题](https://www.bilibili.com/video/xxx)
```
