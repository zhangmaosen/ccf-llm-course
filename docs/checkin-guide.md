# 📅 学习打卡指南

## 📝 如何打卡？

### 快速打卡步骤

1. 点击仓库的 **Issues** 标签
2. 点击 **New Issue**
3. 选择 **📅 每日打卡** 模板
4. **填写日期**（格式：YYYY-MM-DD，如：2026-03-17）
5. 填写姓名和学习内容
6. 提交 Issue

**快捷链接：** [立即打卡](https://github.com/zhangmaosen/ccf-llm-course/issues/new?template=checkin.yml)

### 日期填写说明

| 格式 | 示例 | 说明 |
|------|------|------|
| ✅ 正确 | 2026-03-17 | 年-月-日，用短横线分隔 |
| ❌ 错误 | 2026/03/17 | 不要用斜线 |
| ❌ 错误 | 2026.03.17 | 不要用点号 |

## 打卡规则

### 基本规则

| 规则 | 说明 |
|-----|------|
| 每日一卡 | 每天最多打卡1次 |
| 真实记录 | 记录真实的学习内容和时长 |
| 持续坚持 | 建议连续打卡21天养成习惯 |

| 行为 | 积分 |

| 每日打卡 | +1 |

## 常见问题

**Q: 一天可以打卡多次吗？**
A: 每天最多打卡1次，建议在当天学习结束后打卡。

**Q: 忘记打卡怎么办？**
A: 可以补卡，但建议养成每日打卡的习惯。

**Q: 打卡记录在哪里看？**
A: 查看 Issues 中带有 `checkin` 标签的内容。

**Q: 如何获得更多积分？**
A: 多学习、多分享、多提问，坚持连续打卡！
