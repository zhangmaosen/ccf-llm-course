# CCF 大模型能力认证课程大纲（中小学版）

## 课程定位

本课程基于 CCF（中国计算机学会）大模型能力认证大纲中的中小学组知识点表设计，旨在帮助中小学生系统性地了解人工智能和大语言模型的基础知识，培养科技创新意识和计算思维能力。

## 教学目标

### 知识目标
- 掌握人工智能的基本概念、发展历程和应用领域
- 理解大语言模型的基本原理和核心能力
- 了解模型训练、评估和部署的基本流程

### 能力目标
- 能够使用 AI 工具解决实际问题
- 能够设计有效的提示词（Prompt）
- 具备基础的 AI 伦理判断能力

### 素养目标
- 培养科学探索精神
- 建立正确的科技伦理观念
- 提升信息素养和批判性思维

## 知识模块详解

### 模块一：人工智能基础概念

**知识点：**
| 子模块 | 知识点 |
|--------|--------|
| 人工智能相关概念定义 | 智能、人工智能、机器学习、模型、训练 |
| 机器学习流程及经典模型 | 数据预处理、模型训练、验证、测试、神经网络 |
| 验证及评测 | 交叉验证、精确率、召回率、受试者工作特征曲线 |
| 人工智能常见应用领域 | 自然语言处理、计算机视觉 |

**教学重点：**
- 什么是人工智能？从生活中的例子引入
- 机器如何"学习"？用简单比喻解释训练过程
- 如何判断 AI 做得好不好？评测指标的意义

---

### 模块二：大模型基础概念

**知识点：**
| 子模块 | 知识点 |
|--------|--------|
| 自然语言的基础概念 | 语言的定义、语言的特点、语言的应用 |
| 基本定义 | 大语言模型的定义、核心范式（生成、理解） |
| 发展历程与现状 | 四代语言模型的发展历程 |

**教学重点：**
- 语言是人类最重要的交流工具
- 大语言模型为什么"大"？参数量的概念
- 从统计模型到神经网络模型的发展脉络

---

### 模块三：模型架构

**知识点：**
| 子模块 | 知识点 |
|--------|--------|
| 注意力机制 | 查询（query）- 键（key）- 值（value），softmax 函数 |
| 主流架构 | 编码器-解码器架构、因果解码器架构、前缀解码器架构 |

**教学重点：**
- 注意力机制：模型如何"关注"重要信息
- Transformer：现代大模型的基础架构
- 用动画和图示帮助学生理解复杂概念

---

### 模块四：预训练技术

**知识点：**
| 子模块 | 知识点 |
|--------|--------|
| 回归分析 | 极大似然学习 |
| 自监督学习 | 对比学习、掩码预测、自回归 |
| 预训练任务 | 下一个词元预测（语言建模）、去噪自编码（如 BERT 的 MLM） |
| 基于批次数据的训练方法 | Batch 及 Batch Size 概念 |
| 监督学习 | 监督学习、无监督学习 |

**教学重点：**
- 预训练：让模型"读"海量文本学习语言规律
- 自监督学习：不需要人工标注的学习方式
- 用"猜下一个词"游戏帮助学生理解语言建模

## 教学建议

### 教学方法
1. **概念可视化**：使用图示、动画解释抽象概念
2. **案例驱动**：用生活中的例子引入知识点
3. **动手实践**：设计简单有趣的编程实验
4. **讨论反思**：引导学生思考 AI 的社会影响

### 资源配置
- 每个模块配备 PPT 讲义
- 提供配套的练习题和答案
- 推荐在线实验平台和工具

### 评价方式
- 课堂参与和讨论
- 练习题完成情况
- 小组项目展示
- 期末知识测验

## 附录：成人组知识点对比

成人组在以下方面有更深入的要求：
- 扩展法则（KM 扩展法则、Chinchilla 扩展法则）
- Transformer 详细配置和优化
- 分布式训练技术
- 参数高效微调方法（LoRA 等）
- 强化学习对齐方法
- 多智能体系统

中小学版课程注重概念理解和实践体验，不涉及复杂的数学推导和编程实现。
