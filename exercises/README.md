## 📖 题目格式示例

### 选择题格式

每道选择题采用以下格式，每个选项单独一行，方便阅读：

```
### 1. 题目内容？

A. 选项A
B. 选项B
C. 选项C
D. 选项D

<details>
<summary>💡 点击查看答案</summary>

**答案：X**

解析内容...
</details>
```

### 填空题格式

使用下划线标注填空位置：

```
### 5. 机器学习的三要素是 \_\_\_\_\_\_、\_\_\_\_\_\_ 和 \_\_\_\_\_\_。
```

## 📖 使用说明

### 测试题结构

每份测试题包含：
- **选择题**（20分）：考查基础概念
- **填空题**（20分）：考查关键术语
- **简答题**（30分）：考查理解深度
- **应用题**（30分）：考查实际应用能力

### 如何使用

1. **学完课程后完成测试**
2. **独立完成，不看答案**
3. **完成后点击展开查看答案**
4. **记录错误，复习相关内容**

### 评分标准

| 分数 | 评价 | 建议 |
|:----:|:----:|------|
| 90-100 | ⭐⭐⭐⭐⭐ 优秀 | 可以学习下一课 |
| 70-89 | ⭐⭐⭐⭐ 良好 | 基本掌握，继续深入 |
| 60-69 | ⭐⭐⭐ 及格 | 建议复习本课内容 |
| <60 | 需要重学 | 重新学习本课内容 |

## 📚 练习建议

1. **分步学习**：每学完一课就完成对应测试
2. **错题回顾**：错误题目要回到课程复习
3. **讨论交流**：可以和同学讨论难题
4. **实践应用**：尝试将知识应用到实际项目中
