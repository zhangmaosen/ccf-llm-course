### 2. 精确率的公式是？

- A. TP / (TP + FP)

## 三、计算题（每题15分，共30分）

### 7. 一个垃圾邮件检测系统，对100封邮件进行分类：

- 实际是垃圾邮件：20封
- 预测为垃圾邮件：25封
- 正确识别的垃圾邮件：18封

请计算：
1. 混淆矩阵中的 TP、FP、FN、TN
2. 精确率、召回率、准确率

<details>
<summary>💡 点击查看参考答案</summary>

**答案：**

**第1步：计算混淆矩阵**

已知：
- 实际为正（垃圾邮件）= 20封
- 预测为正（垃圾邮件）= 25封
- TP（正确识别垃圾邮件）= 18封

计算：
- FP = 预测为正 - TP = 25 - 18 = 7封
- FN = 实际为正 - TP = 20 - 18 = 2封
- TN = 总数 - TP - FP - FN = 100 - 18 - 7 - 2 = 73封

**混淆矩阵：**

| | 预测为正 | 预测为负 |
|-|---------|---------|
| 实际为正 | TP=18 | FN=2 |
| 实际为负 | FP=7 | TN=73 |

**第2步：计算指标**

- 精确率 = TP / (TP + FP) = 18 / 25 = **72%**
- 召回率 = TP / (TP + FN) = 18 / 20 = **90%**
- 准确率 = (TP + TN) / 总数 = (18 + 73) / 100 = **91%**
</details>

### 8. 在医疗诊断场景，精确率和召回率哪个更重要？为什么？

<details>
<summary>💡 点击查看参考答案</summary>

**参考答案：**

**召回率更重要。**

**原因分析：**

| 指标 | 含义 | 错误后果 |
|------|------|---------|
| 召回率低 | 漏诊（有病没查出来） | 延误治疗，危及生命 |
| 精确率低 | 误诊（没病说有病） | 进一步检查排除 |

**医疗场景特点：**
- 漏诊的后果比误诊更严重
- 宁可多检查，不能漏掉病人
- 所以前优先保证召回率

**实际应用：**
- 癌症筛查：召回率优先
- 普通体检：两者平衡

**不同场景的优先级：**

| 场景 | 优先指标 | 原因 |
|------|:-------:|------|
| 癌症筛查 | 召回率 | 不能漏诊 |
| 垃圾邮件 | 精确率 | 不能误判正常邮件 |
| 安检 | 召回率 | 不能漏掉危险品 |
| 推荐系统 | 精确率 | 推荐要准确 |
</details>

## 四、应用题（30分）

### 9. 你需要评测一个情感分析模型，设计一个评测方案：

要求：
1. 选择评测指标
2. 设计评测数据集
3. 说明评测流程

<details>
<summary>💡 点击查看参考答案</summary>

**参考答案：**

**1. 评测指标选择**

| 指标 | 原因 |
|------|------|
| 准确率 | 整体正确率 |
| 精确率 | 正向预测的准确性 |
| 召回率 | 找出所有正向的能力 |
| F1分数 | 综合指标 |

**2. 评测数据集设计**

```
数据来源：
- 公开数据集（如微博情感数据）
- 人工标注数据
- 多领域覆盖（商品评论、新闻、社交媒体）

数据构成：
- 正向情感：30%
- 负向情感：30%
- 中性情感：40%

数据量：1000-5000条
```

**3. 评测流程**

```
第1步：数据准备
├── 收集评测数据
├── 人工标注标签
└── 划分测试集

第2步：模型预测
├── 输入测试文本
├── 获取模型预测结果
└── 记录预测标签

第3步：计算指标
├── 计算混淆矩阵
├── 计算各项指标
└── 分类别分析

第4步：错误分析
├── 找出错误案例
├── 分析错误原因
└── 提出改进建议

第5步：生成报告
├── 整体性能
├── 各类别性能
└── 改进建议
```
</details>
