## 📖 第一部分：什么是人工智能？

### 1.1 智能：人类 vs 机器

**人类的智能表现：**

| 能力 | 例子 |
|-----|------|
| 🧠 思考 | 看到乌云，知道要下雨了 |
| 👀 识别 | 一眼认出好朋友 |
| 💬 交流 | 用语言表达想法 |
| 🎯 解决问题 | 迷路时用地图导航 |

**人工智能（Artificial Intelligence，AI）**：
> 让机器模拟人类的智能行为，能够像人一样"思考"、"学习"和"解决问题"。

### 1.2 AI 就在身边

**你每天都在用 AI：**

```
📱 手机解锁 → 人脸识别 AI
🎵 音乐推荐 → 推荐系统 AI  
🗺️ 导航软件 → 路径规划 AI
🗣️ 语音助手 → 语音识别 AI
📧 垃圾邮件过滤 → 分类 AI
```

### 1.3 AI 发展简史

```
1950s ──→ 1980s ──→ 2010s ──→ 2020s ──→ 现在
  │         │         │         │         │
图灵测试   专家系统   深度学习   AlphaGo   ChatGPT
"机器能    "规则    "让机器    "围棋     "对话
思考吗？"   决策"    自己学"    冠军"     AI"
```

**重要里程碑：**
- **1950年**：图灵提出"机器能思考吗？"
- **1997年**：IBM 深蓝击败国际象棋冠军
- **2016年**：AlphaGo 击败围棋世界冠军
- **2022年**：ChatGPT 发布，AI 走进千家万户

## 📖 第二部分：机器如何学习？

### 2.1 人类学习 vs 机器学习

**人类学习过程：**
```
婴儿学走路：
1. 👀 观察：看大人怎么走
2. 🚶 尝试：扶着东西走
3. 💪 练习：摔倒，爬起来
4. ✅ 掌握：终于会走了！
```

**机器学习过程：**
```
机器学识别猫：
1. 📊 数据：看10万张猫的照片
2. 🔢 特征：学习猫的特征（耳朵、胡须、尾巴）
3. 🔄 训练：不断调整，减少错误
4. ✅ 掌握：能识别出猫了！
```

### 2.2 机器学习的三要素

```
┌─────────────────────────────────────────┐
│           机器学习三要素                  │
├─────────────────────────────────────────┤
│                                         │
│   📊 数据    →  学习材料                 │
│   🧠 算法    →  学习方法                 │
│   💻 算力    →  学习能力                 │
│                                         │
└─────────────────────────────────────────┘
```

### 2.3 机器学习的基本流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  收集数据 │ ─→ │ 数据预处理 │ ─→ │  训练模型 │ ─→ │  测试验证 │
└──────────┘    └──────────┘    └──────────┘    └──────────┘
     │               │               │               │
     ▼               ▼               ▼               ▼
 照片、文本      清洗、整理      让机器学习      检查效果
 视频、音频      标注、分类      找规律          优化改进
```

**举个例子：识别手写数字**

| 步骤 | 做什么 | 例子 |
|-----|-------|------|
| 收集数据 | 收集手写数字图片 | 6万张0-9的手写图片 |
| 数据预处理 | 统一大小、归一化 | 把图片都变成28×28像素 |
| 训练模型 | 让机器学习特征 | 学习每个数字的笔画特征 |
| 测试验证 | 检查准确率 | 用新图片测试，看对了多少 |

## 📖 第三部分：监督学习 vs 无监督学习

### 3.1 监督学习：有老师带着学

**特点：** 有标签（答案），老师告诉机器对错

**类比：做作业有答案**

```
老师给题目：这是猫吗？（有答案：是/否）
机器回答：是！
老师反馈：✅ 正确！
机器记住：有胡须、尖耳朵的可能是猫
```

**应用场景：**

| 任务 | 输入 | 输出 |
|-----|-----|-----|
| 图像分类 | 猫的照片 | "猫" |
| 垃圾邮件检测 | 邮件内容 | "垃圾"/"正常" |
| 房价预测 | 房屋信息 | 价格 |

### 3.2 无监督学习：自己探索发现

**特点：** 没有标签，机器自己找规律

**类比：自己整理房间**

```
没有老师告诉你怎么整理
你自己发现：
- 书放一起
- 衣服放一起  
- 玩具放一起
```

**应用场景：**

| 任务 | 说明 |
|-----|------|
| 客户分群 | 把相似客户分成不同群体 |
| 异常检测 | 发现异常的交易行为 |
| 推荐系统 | 发现相似的商品 |

### 3.3 对比总结

```
        监督学习                    无监督学习
           │                           │
    ┌──────┴──────┐             ┌──────┴──────┐
    │ 有标签      │             │ 无标签      │
    │ 有老师指导  │             │ 自己探索    │
    │ 预测结果    │             │ 发现模式    │
    └─────────────┘             └─────────────┘
    
    例子：分类、回归            例子：聚类、降维
```

## 📖 第四部分：神经网络基础

### 🎬 视频导学
📺 [3Blue1Brown - 神经网络系列](https://www.bilibili.com/video/BV1bx411M7Zx)（强烈推荐！动画超赞）

### 4.1 模仿人脑

**人脑神经元：**
```
    树突（接收信号）
      │
      ▼
   ┌──────┐
   │ 细胞 │ ──→ 轴突（传递信号）
   │ 体   │
   └──────┘
```

**人工神经元：**
```
    输入
      │
      ▼
   ┌──────┐
   │ 计算 │ ──→ 输出
   │ 单元 │
   └──────┘
```

**人工神经元：**
```
    输入
      │
      ▼
   ┌──────┐
   │ 计算 │ ──→ 输出
   │ 单元 │
   └──────┘
```

### 4.2 神经网络结构

```
输入层          隐藏层          输出层
 (图片)    (特征提取/计算)      (结果)
    
  ○ ──────────→ ○ ──────────→ ○
  ○ ──────────→ ○ ──────────→ ○
  ○ ──────────→ ○ ──────────→ ○
  ○ ──────────→ ○ ──────────→ ○
  
  像素值        中间特征        分类结果
```

**简单理解：**
- **输入层**：接收原始数据（如图片的像素）
- **隐藏层**：提取特征、进行计算
- **输出层**：给出最终结果

## 📖 第五部分：如何评价 AI 的表现？

### 5.1 基本评测指标

**问题场景：** 有100封邮件，其中20封是垃圾邮件

| 预测\实际 | 垃圾邮件 | 正常邮件 |
|----------|---------|---------|
| 预测为垃圾 | 18 (TP) | 5 (FP) |
| 预测为正常 | 2 (FN) | 75 (TN) |

**核心指标：**

| 指标 | 公式 | 含义 | 本例 |
|-----|------|------|------|
| 准确率 | 正确数/总数 | 整体对不对 | (18+75)/100 = 93% |
| 精确率 | TP/(TP+FP) | 预测垃圾的有多少真是垃圾 | 18/23 = 78% |
| 召回率 | TP/(TP+FN) | 真垃圾的有多少被找出来 | 18/20 = 90% |

### 5.2 生活中的例子

**医疗诊断：**
- 精确率高 → 诊断有病的，真的有病（避免误诊）
- 召回率高 → 有病的都能被诊断出来（避免漏诊）

**垃圾邮件：**
- 精确率高 → 别把正常邮件当垃圾
- 召回率高 → 垃圾邮件都拦截

## 📖 第六部分：AI 的应用领域

### 6.1 自然语言处理（NLP）

**让机器理解和生成人类语言**

```
输入："今天天气怎么样？"
处理：理解意图、检索信息
输出："北京今天晴，气温15-25℃"
```

**应用案例：**
- 🌐 机器翻译（Google翻译、DeepL）
- 💬 智能客服
- 📝 文本摘要
- 🎭 情感分析

### 6.2 计算机视觉（CV）

**让机器"看懂"图像和视频**

```
输入：一张猫的照片
处理：提取特征、识别形状
输出："这是一只猫"
```

**应用案例：**
- 📷 人脸识别（手机解锁）
- 🚗 自动驾驶
- 🏥 医学影像诊断
- 👕 商品识别

### 6.3 其他重要领域

| 领域 | 说明 | 应用 |
|-----|------|------|
| 推荐系统 | 个性化推荐 | 抖音、淘宝推荐 |
| 语音识别 | 听懂语音 | Siri、小爱同学 |
| 游戏AI | 游戏策略 | AlphaGo、游戏NPC |
| 机器人 | 物理行动 | 扫地机器人、配送机器人 |
