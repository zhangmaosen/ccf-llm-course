> 让 AI 从海量数据中学习

## 🎯 学习目标

- 理解预训练的概念
- 了解自监督学习方法
- 认识训练过程的关键要素

---

## 📖 课程内容

### 一、什么是预训练？

#### 类比：人类的学习

**婴儿学说话**：
1. 听父母说话（大量输入）
2. 模仿发音（尝试输出）
3. 纠正错误（不断调整）
4. 掌握语言（技能形成）

**模型预训练**：
1. 读取海量文本
2. 预测下一个词
3. 对比正确答案
4. 调整模型参数

---

### 二、预训练任务

#### 1. 下一个词元预测

**游戏：猜下一个词**

```
"今天天气真" → ?
可能的答案：好、不错、糟糕...
```

模型通过大量这样的练习，学习语言的规律。

#### 2. 掩码语言建模（MLM）

**游戏：填空题**

```
"我 喜欢 吃 苹果" 
模型需要猜出被遮住的词
```

BERT 使用这种方法训练。

---

### 三、自监督学习

#### 核心思想

不需要人工标注，数据本身提供监督信号。

| 学习方式 | 需要标注吗 | 例子 |
|---------|----------|------|
| 监督学习 | 需要 | 图像分类（需要标签） |
| 自监督学习 | 不需要 | 预测下一个词 |
| 无监督学习 | 不需要 | 聚类、降维 |

#### 自监督学习类型

1. **对比学习**：区分相似和不同的样本
2. **掩码预测**：猜被遮住的内容
3. **自回归**：根据前面预测后面

---

### 四、训练的关键要素

#### Batch 和 Batch Size

**类比：做作业**
- 一次做 1 道题 → Batch Size = 1
- 一次做 10 道题 → Batch Size = 10

**Batch Size 的影响**：
- 太小：学习慢
- 太大：需要更多内存

#### 学习率

**类比：调节步子大小**
- 学习率大：步子大，可能跳过最优解
- 学习率小：步子小，学习慢但稳定

#### 优化器

常用的优化器：**Adam**、**SGD**

优化器的作用：帮助模型找到最优参数。

---

### 五、监督学习 vs 无监督学习

| | 监督学习 | 无监督学习 |
|-|---------|-----------|
| 数据 | 有标签 | 无标签 |
| 目标 | 预测标签 | 发现模式 |
| 例子 | 分类、回归 | 聚类、降维 |
