## 📖 课程内容

### 一、什么是解码？

#### 生成文本的过程

模型输出的是**概率分布**：

```
输入："今天天气"
模型输出概率：{好: 0.6, 差: 0.2, 热: 0.15, 冷: 0.05}
```

解码就是从这个概率分布中选择词。

---

### 二、解码方法

#### 1. 贪心搜索

**方法**：每次选择概率最高的词

```
输入：今天天气
选择：好 (概率最高)
继续：今天天气好
...
```

**优点**：速度快
**缺点**：可能错过更好的答案

#### 2. 束搜索（Beam Search）

**方法**：保留多个候选，最后选最好的

```
保留 3 个候选：
1. 今天天气很好
2. 今天天气不错
3. 今天天气真好
```

**优点**：质量更好
**缺点**：计算量大

---

### 三、从模型到应用

#### 部署流程

```
训练好的模型 → 优化 → 部署 → 服务用户
```

#### 部署方式

| 方式 | 特点 |
|-----|------|
| 云端部署 | 模型在服务器，用户通过网络访问 |
| 本地部署 | 模型在用户设备上运行 |
| 混合部署 | 结合两者优势 |

#### 推理加速

让模型运行更快的技术：
- **量化**：降低精度，减少计算量
- **剪枝**：删除不重要的参数
- **蒸馏**：用小模型学习大模型
