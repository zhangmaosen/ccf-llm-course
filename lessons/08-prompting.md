# 第八课：提示学习

> 学会和 AI 对话

## 🎯 学习目标

- 理解提示工程的概念
- 掌握提示设计技巧
- 学会使用 AI API

---

## 📖 课程内容

### 一、什么是提示学习？

#### 提示（Prompt）的作用

**提示**：给 AI 的指令或输入

好的提示 = 好的回答
差的提示 = 差的回答

#### 提示学习的目的

- 让模型更好地理解任务
- 激发模型的潜在能力
- 不需要修改模型参数

---

### 二、提示设计技巧

#### 技巧 1：明确任务

❌ 差的提示：
```
写一篇作文
```

✅ 好的提示：
```
请写一篇关于"春天的校园"的作文，字数 500 字左右，要求使用描写和抒情的表达方式。
```

#### 技巧 2：提供示例

```
任务：判断情感倾向

示例 1：
句子：今天天气真好！
情感：积极

示例 2：
句子：我很失望。
情感：消极

请判断以下句子的情感：
句子：这部电影太精彩了！
情感：
```

#### 技巧 3：角色扮演

```
你是一位经验丰富的数学老师，请用通俗易懂的方式解释什么是"函数"。
```

#### 技巧 4：分步骤要求

```
请按以下步骤帮我分析这篇文章：
1. 总结文章主旨
2. 列出关键观点
3. 提出你的看法
```

---

### 三、常见模型 API 使用

#### 基本使用流程

```python
# 伪代码示例
response = model.chat(
    messages=[
        {"role": "user", "content": "你好"}
    ]
)
print(response)
```

#### 重要参数

| 参数 | 作用 |
|-----|------|
| temperature | 控制随机性（0-2） |
| max_tokens | 限制输出长度 |
| top_p | 控制采样范围 |

---

### 四、提示的局限性

**提示学习不能做的**：
- 改变模型的知识
- 让模型完全记住长文本
- 解决超出模型能力的问题

## 🔬 课堂实验

### 实验 1：提示优化

优化以下提示，让回答更准确：

原提示：
```
介绍一下中国
```

优化后的提示：
```
（请学生填写）
```

### 实验 2：体验不同参数

尝试不同的 temperature 值，观察生成结果的差异：
- temperature = 0.1（很确定）
- temperature = 1.0（平衡）
- temperature = 1.8（很有创意）
