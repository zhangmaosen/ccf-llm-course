## 📖 课程内容

### 一、什么是推理？

#### 推理的定义

**推理**：从已知信息得出新结论的思维过程。

#### 推理的类型

| 类型 | 特点 | 例子 |
|-----|------|------|
| 演绎推理 | 从一般到特殊 | 所有人都会死 → 苏格拉底是人 → 苏格拉底会死 |
| 归纳推理 | 从特殊到一般 | 看到很多天鹅是白的 → 天鹅是白色的 |
| 类比推理 | 从相似性出发 | A 和 B 相似，A 有属性 X → B 也有属性 X |

---

### 二、思维链（Chain of Thought）

#### 直接回答 vs 思维链

**问题**：
```
小明有 5 个苹果，给了小红 2 个，又买了 3 个，请问小明现在有几个苹果？
```

**直接回答**：
```
答：6 个
```

**思维链**：
```
第一步：小明原有 5 个苹果
第二步：给了小红 2 个，还剩 5 - 2 = 3 个
第三步：又买了 3 个，现在有 3 + 3 = 6 个
答：6 个
```

#### 思维链的作用

- ✅ 提高复杂问题的准确率
- ✅ 展示推理过程
- ✅ 便于发现错误
