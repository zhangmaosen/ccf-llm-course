## 📖 课程内容

### 一、什么是智能体？

#### 定义

**智能体（Agent）**：能够感知环境、做出决策、执行行动的 AI 系统。

#### 与聊天机器人的区别

| | 聊天机器人 | 智能体 |
|-|-----------|--------|
| 能力 | 对话 | 对话 + 行动 |
| 自主性 | 被动响应 | 主动执行 |
| 记忆 | 有限 | 持久记忆 |
| 工具 | 无 | 可使用工具 |

---

### 二、智能体的核心组件

```
         ┌─────────────┐
         │   Profile   │ ← 身份设定
         └─────────────┘
               │
         ┌─────────────┐
         │   Memory    │ ← 记忆系统
         └─────────────┘
               │
         ┌─────────────┐
         │  Planning   │ ← 规划能力
         └─────────────┘
               │
         ┌─────────────┐
         │   Action    │ ← 执行行动
         └─────────────┘
```

#### 1. Profile（身份设定）

定义智能体的角色、性格、能力范围。

```
你是一个乐于助人的学习助手，擅长解答数学问题，态度友善。
```

#### 2. Memory（记忆）

- **短期记忆**：当前对话内容
- **长期记忆**：历史交互和知识

#### 3. Planning（规划）

分解复杂任务，制定行动计划。

#### 4. Tools（工具）

使用外部工具扩展能力：
- 🔍 搜索引擎
- 📊 数据分析
- 🌐 网页访问

---

### 三、智能体应用案例

#### WebGPT

能够浏览网页、搜索信息、给出回答的智能体。

#### 斯坦福小镇

25 个 AI 智能体在虚拟小镇中生活、社交、工作。

#### 个人助手

帮助安排日程、管理任务、回答问题。
