## 🚀 课前思考

**聊天机器人 vs 智能助手 vs 智能体**

```
┌─────────────────────────────────────────────────┐
│              聊天机器人                          │
├─────────────────────────────────────────────────┤
│ 只能聊天，不能做事                              │
│ 用户："帮我订张机票"                           │
│ 机器人："抱歉，我无法为您预订机票"             │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│              智能助手                            │
├─────────────────────────────────────────────────┤
│ 能做简单的预设任务                              │
│ 用户："帮我订张机票"                           │
│ 助手：调用预订系统，完成预订                    │

┌─────────────────────────────────────────────────┐
│              智能体（Agent）                     │
├─────────────────────────────────────────────────┤
│ 能自主规划、决策、执行                          │
│ 用户："帮我计划一次旅行"                       │
│ 智能体：                                        │
│  1. 搜索目的地信息                              │
│  2. 比较机票价格                                │
│  3. 推荐酒店                                    │
│  4. 制定行程                                    │
│  5. 预订所有项目                                │
└─────────────────────────────────────────────────┘
```

## 📖 第一部分：什么是智能体？

### 🎬 视频导学
📺 [AI Agent 详解](https://www.bilibili.com/video/BV1xxx)
🔗 [LangChain 官方教程](https://python.langchain.com/docs/get_started/introduction)

### 1.1 智能体的定义

```
智能体（Agent）= 能够感知环境、做出决策、执行行动的 AI 系统

核心特征：
────────────────
✅ 自主性：不需要人类一步步指挥
✅ 感知能力：能够获取信息
✅ 决策能力：能够规划行动
✅ 行动能力：能够执行任务
```

### 1.2 智能体 vs 传统 AI

```
┌─────────────┬─────────────────┬─────────────────┐
│             │    传统 AI      │     智能体      │
├─────────────┼─────────────────┼─────────────────┤
│   交互方式  │   一问一答      │   自主执行      │
├─────────────┼─────────────────┼─────────────────┤
│   行动能力  │   只能说话      │   能调用工具    │
├─────────────┼─────────────────┼─────────────────┤
│   记忆      │   对话内有限    │   持久记忆      │
├─────────────┼─────────────────┼─────────────────┤
│   规划      │   被动响应      │   主动规划      │
├─────────────┼─────────────────┼─────────────────┤
│   例子      │   ChatGPT       │   AutoGPT       │
└─────────────┴─────────────────┴─────────────────┘
```

### 1.3 智能体的发展

```
2022：ChatGPT 发布
      └── 只能聊天

2023：AutoGPT、BabyAGI
      └── 早期智能体实验

2024：多智能体系统
      └── 智能体协作

2025+：GUI 智能体
      └── 像人一样操作电脑
```

## 📖 第二部分：智能体的核心组件

### 2.1 智能体架构

```
┌─────────────────────────────────────────────────┐
│                 智能体架构                       │
├─────────────────────────────────────────────────┤
│                                                 │
│  ┌─────────────┐                               │
│  │   Profile   │ ← 身份设定                    │
│  │  (角色)     │                               │
│  └─────────────┘                               │
│         ↓                                       │
│  ┌─────────────┐                               │
│  │   Memory    │ ← 记忆系统                    │
│  │  (记忆)     │                               │
│  └─────────────┘                               │
│         ↓                                       │
│  ┌─────────────┐                               │
│  │  Planning   │ ← 规划模块                    │
│  │  (规划)     │                               │
│  └─────────────┘                               │
│         ↓                                       │
│  ┌─────────────┐                               │
│  │   Tools     │ ← 工具调用                    │
│  │  (工具)     │                               │
│  └─────────────┘                               │
│         ↓                                       │
│  ┌─────────────┐                               │
│  │   Action    │ ← 执行行动                    │
│  │  (行动)     │                               │
│  └─────────────┘                               │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 2.2 Profile（身份设定）

```
定义智能体是谁、擅长什么

例子：
────────────────
名称：小明助手
角色：学习辅导专家
能力：数学、物理、化学
性格：耐心、鼓励型
语言：亲切、简单

设定方式：
────────────────
"你是一位耐心的小学数学老师，擅长用简单的语言
解释复杂的概念，会给学生鼓励，引导他们思考。"
```

### 2.3 Memory（记忆）

```
┌─────────────────┬───────────────────────────────┐
│     记忆类型    │              说明              │
├─────────────────┼───────────────────────────────┤
│  短期记忆       │ 当前对话的上下文              │
│                 │ 例：刚才说了什么              │
├─────────────────┼───────────────────────────────┤
│  长期记忆       │ 持久存储的信息                │
│                 │ 例：用户的偏好、历史交互      │
├─────────────────┼───────────────────────────────┤
│  工作记忆       │ 当前任务相关信息              │
│                 │ 例：正在处理的问题            │
└─────────────────┴───────────────────────────────┘
```

### 2.4 Tools（工具）

```
智能体可以使用的"手"：

┌─────────────────┬───────────────────────────────┐
│      工具       │              用途              │
├─────────────────┼───────────────────────────────┤
│  搜索引擎       │ 获取最新信息                  │
├─────────────────┼───────────────────────────────┤
│  代码执行       │ 运行程序、计算                │
├─────────────────┼───────────────────────────────┤
│  数据库查询     │ 访问数据                      │
├─────────────────┼───────────────────────────────┤
│  API 调用       │ 访问外部服务                  │
├─────────────────┼───────────────────────────────┤
│  文件操作       │ 读写文件                      │
└─────────────────┴───────────────────────────────┘
```

### 2.5 Planning（规划）

```
任务分解示例：

任务：帮我写一篇关于 AI 的报告

智能体规划：
────────────────
第1步：搜索 AI 最新发展
第2步：整理 AI 应用案例
第3步：分析 AI 的影响
第4步：撰写报告大纲
第5步：填充各部分内容
第6步：检查和修改
```

## 📖 第三部分：智能体的应用

### 3.1 WebGPT

```
能够浏览网页、搜索信息的智能体

能力：
────────────────
1. 理解用户问题
2. 自动搜索相关网页
3. 阅读和理解网页内容
4. 整合信息，给出回答
5. 引用来源

例子：
────────────────
用户："最新的 iPhone 有什么新功能？"
WebGPT：搜索 → 阅读 → 整合 → 回答 + 引用
```

### 3.2 代码智能体

```
能够编写、调试、运行代码的智能体

能力：
────────────────
1. 理解编程需求
2. 生成代码
3. 运行和测试
4. 发现和修复 bug
5. 优化代码

例子：
────────────────
用户："帮我写一个网页爬虫"
智能体：设计 → 编写 → 测试 → 调试 → 交付
```

### 3.3 个人助手

```
能够帮助管理日常事务的智能体

能力：
────────────────
1. 安排日程
2. 管理任务
3. 回答问题
4. 提醒事项
5. 自动化工作流

例子：
────────────────
用户："安排明天下午的会议"
智能体：查日历 → 选时间 → 发邀请 → 设提醒
```

## 📖 第四部分：多智能体协作

### 4.1 为什么需要多智能体？

```
单个智能体的局限：
────────────────
- 能力有限
- 可能出错
- 效率不高

多智能体协作：
────────────────
- 分工合作
- 相互检查
- 效率更高
```

### 4.2 协作模式

```
┌─────────────────────────────────────────────────┐
│                 多智能体协作                     │
├─────────────────────────────────────────────────┤
│                                                 │
│  模式1：流水线                                  │
│  ──────────────                                 │
│  智能体A → 智能体B → 智能体C                   │
│  （研究）  （写作）  （审核）                   │
│                                                 │
│  模式2：辩论                                    │
│  ──────────────                                 │
│  智能体A ←→ 智能体B                             │
│  （正方）    （反方）                           │
│       ↓                                         │
│  裁判智能体                                     │
│                                                 │
│  模式3：分工                                    │
│  ──────────────                                 │
│       ┌→ 智能体A（前端）                        │
│  任务 ─→ 智能体B（后端）                        │
│       └→ 智能体C（测试）                        │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 4.3 斯坦福小镇

```
著名的多智能体实验：

设定：
────────────────
25 个 AI 智能体在一个虚拟小镇生活

每个智能体：
────────────────
- 有自己的身份和性格
- 有记忆和关系
- 能自主行动
- 能与其他智能体交流

有趣的发现：
────────────────
- 智能体自发组织活动
- 形成社交关系
- 记住彼此的对话
- 产生"涌现行为"
```

### 3.4 OpenClaw - 个人 AI 助手

```
2025 年爆火的个人 AI 助手平台

官网：https://openclaw.ai/

什么是 OpenClaw？
────────────────
OpenClaw 是一个让每个人都能拥有自己的 AI 助手的平台。
你可以通过多种方式与 AI 助手交互：
- Telegram、Discord、WhatsApp 等聊天应用
- 语音对话
- 网页界面

核心能力：
────────────────
✅ 多模态交互：文字、语音、图片
✅ 工具调用：搜索、代码执行、文件操作
✅ 记忆系统：记住用户偏好和历史对话
✅ 多平台：随时随地访问
✅ 自动化：定时任务、提醒

为什么爆火？
────────────────
1. 低门槛：不需要编程知识
2. 实用性强：真正能帮人做事
3. 个性化：可以定制自己的助手
4. 跨平台：支持多种聊天软件

例子：
────────────────
用户："帮我总结今天的新闻"
OpenClaw：搜索 → 阅读 → 总结 → 发送到 Telegram

用户："提醒我每天早上 8 点吃药"
OpenClaw：创建定时任务 → 每天准时提醒

用户："帮我分析这张图片"
OpenClaw：接收图片 → AI 分析 → 返回结果
```
