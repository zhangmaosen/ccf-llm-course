# 第十一课：模型评测

> 如何评价 AI 的能力

## 🎯 学习目标

- 理解模型评测的意义
- 掌握常见评测指标
- 了解评测方法

---

## 📖 课程内容

### 一、为什么要评测？

#### 评测的目的

- 比较不同模型的性能
- 发现模型的优缺点
- 指导模型改进方向

#### 评测的基本流程

```
准备测试数据 → 模型预测 → 计算指标 → 分析结果
```

---

### 二、常见评测指标

#### 分类任务指标

| 指标 | 公式 | 含义 |
|-----|------|------|
| 准确率 | 正确预测数 / 总数 | 整体正确率 |
| 精确率 | TP / (TP + FP) | 预测为正的有多少是对的 |
| 召回率 | TP / (TP + FN) | 实际为正的有多少被找出来 |
| F1 分数 | 2 × (精确率 × 召回率) / (精确率 + 召回率) | 综合指标 |

#### 生成任务指标

| 指标 | 用途 |
|-----|------|
| BLEU | 机器翻译质量 |
| ROUGE | 文本摘要质量 |
| 困惑度（PPL） | 语言模型质量 |

#### 排序任务指标

| 指标 | 用途 |
|-----|------|
| NDCG | 搜索结果排序质量 |
| 成功率 | 推荐是否成功 |

---

### 三、评测方法

#### 1. 基于评测基准

使用标准化的测试集：

| 数据集 | 用途 |
|--------|------|
| MMLU | 多任务语言理解 |
| C-Eval | 中文综合评测 |
| BIG-Bench | 大规模基准测试 |

#### 2. 基于人类评估

让人来评价模型输出：
- 相关性
- 准确性
- 流畅性

#### 3. 基于模型评估

用强大的模型评价其他模型。

---

### 四、泛化能力

#### 什么是泛化？

模型在未见过的数据上的表现能力。

#### 数据集划分

```
训练集（70%） → 训练模型
验证集（15%） → 调整参数
测试集（15%） → 最终评估
```
